2.18 Release Notes

A complete list of issues can be seen in the tracker milestone 2.18 .

This release features Compass version 1.0.3, Bourbon version 4.2.7 and Sass version 3.4.22 running on JRuby 9.0.5.0. Please check the full dependencies to find out about the specific Sass and other transitive dependencies included in this release.

Enhancements and Fixes

  • Update SCSS-Lint to version 0.48.0, please check the SCSS-Lint changelog
  • Update Bourbon to version 4.2.7, please check the Bourbon changelog
